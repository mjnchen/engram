# Engram

Personal AI/LLM knowledge base, maintained with [Claude Code](https://claude.ai/code) using the LLM Wiki pattern.

## Structure

| Folder | Purpose |
|---|---|
| `inbox/` | Raw drops: bookmarks, pastes, unprocessed notes |
| `wiki/concepts/` | Distilled concept notes — techniques, architectures, ideas |
| `wiki/tools/` | AI tools, services, and frameworks |
| `wiki/sources/` | Processed articles, papers, and links |
| `wiki/synthesis/` | Cross-concept essays and summaries |
| `_templates/` | Note templates |

## How it works

Notes are written and maintained by Claude Code. To add new content, drop files or bookmarks into `inbox/` and ask Claude to process them. Notes use `[[wikilink]]` syntax and are compatible with [Obsidian](https://obsidian.md).

Seeded from [Hannibal046/Awesome-LLM](https://github.com/Hannibal046/Awesome-LLM) and [NicholasSpisak/second-brain](https://github.com/NicholasSpisak/second-brain).
