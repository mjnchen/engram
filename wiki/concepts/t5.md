# T5 (Text-to-Text Transfer Transformer)

## Summary

T5 is a Transformer model that frames every NLP task as text-to-text: inputs and outputs are always strings, whether the task is translation, summarization, classification, or QA. You use a single model architecture and training objective across all tasks by prepending a task prefix to the input (e.g., "summarize: ..."). This unified framing makes multi-task learning and transfer straightforward.

## Key Ideas

- Encoder-decoder architecture; the encoder reads input, the decoder generates the output token by token
- Pretraining objective: span corruption (mask random contiguous spans of tokens, predict them in the decoder)
- C4 (Colossal Clean Crawled Corpus) was created specifically for T5 training and became a standard pretraining dataset
- Exhaustive ablations compared objectives, architectures, and data; T5 is as much a systematic study as a model
- FLAN-T5 applied instruction tuning to T5, producing strong instruction-following at modest scale
