# A Survey of Large Language Models (LLMsPracticalGuide)

## Summary

A researcher-maintained curated guide covering the LLM landscape with an organized paper list and evolutionary tree of models. Useful as a living index for navigating the field rather than reading a single static survey paper. Tracks which models descended from which, making architectural lineages visible.

## Key Points

- Organizes papers by topic: pre-training, adaptation, alignment, utilization, and capacity evaluation
- Includes an evolutionary tree diagram showing GPT, BERT, T5, and derivative model lineages
- Distinguishes between models available via API only, open weights, and fully open (data + code)
- Covers practical usage guidance, not just architecture papers
