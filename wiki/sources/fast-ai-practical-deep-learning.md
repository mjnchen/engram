# Practical Deep Learning for Coders (fast.ai)

## Summary

Free 18-hour course by Jeremy Howard and fast.ai. Top-down approach: start with working models, then dig into theory. Covers vision, NLP, tabular data, and generative models using PyTorch and the fastai library.

## Key Points

- Top-down pedagogy — get results first, understand internals later
- Uses PyTorch under the hood; fastai is a high-level wrapper
- Covers fine-tuning pretrained models from the start (before LLMs made this mainstream)
- Free and fully online
