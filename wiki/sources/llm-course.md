# llm-course (mlabonne)

## Summary

Structured learning roadmap for LLMs with runnable Colab notebooks covering the full stack from transformer fundamentals through fine-tuning and deployment. Organized as three sequential tracks: LLM scientist (training/fine-tuning), LLM engineer (building applications), and LLM evaluation.

## Key Points

- Fine-tuning track covers SFT with LoRA, preference alignment with DPO, and merging fine-tuned adapters
- Quantization notebooks cover GGUF (for llama.cpp) and GPTQ/AWQ formats with practical trade-off comparisons
- Evaluation section covers perplexity, benchmark suites, and the LLM-as-judge pattern
- Each notebook is self-contained and runnable on free Colab T4 GPU
