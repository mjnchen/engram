# Build a Large Language Model (From Scratch) — Manning

## Summary

Sebastian Raschka's book guiding readers through implementing, pre-training, and fine-tuning a GPT-style LLM step by step. The book is distinguished by covering the full lifecycle — not just the forward pass but also the data pipeline, training loop, and fine-tuning stages.

## Key Points

- Covers BPE tokenization implementation before building the model, grounding token handling in code rather than abstraction
- Pre-training from scratch (even on small data) gives intuition for why hyperparameter choices matter
- Fine-tuning section covers both instruction tuning and classification fine-tuning with LoRA
- Code is PyTorch and deliberately minimal — no high-level training frameworks, so every step is transparent
