# lm-evaluation-harness

## What it is

Open-source Python framework by EleutherAI for standardized few-shot evaluation of language models across hundreds of tasks.

## What problem it solves

Provides a reproducible, unified benchmark runner so model comparisons are consistent — it powers most open LLM leaderboards including Hugging Face's Open LLM Leaderboard.
