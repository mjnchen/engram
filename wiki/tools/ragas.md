# Ragas

## What it is

Python framework by Exploding Gradients for automated evaluation of RAG (retrieval-augmented generation) pipelines.

## What problem it solves

Evaluating RAG quality end-to-end is hard — Ragas provides LLM-based metrics for faithfulness, answer relevance, context precision, and context recall without requiring human labels.
