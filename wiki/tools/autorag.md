# AutoRAG

## What it is

Automated RAG pipeline optimization tool from Marker Inc Korea that benchmarks different retrieval and generation configurations to find the best setup for your data.

## What problem it solves

Choosing chunking strategies, embedding models, retrievers, and rerankers for a RAG pipeline requires extensive manual experimentation; AutoRAG automates this search with a config-driven approach.
