# Chainlit

## What it is

Chainlit is a Python library by Chainlit for rapidly building and deploying conversational AI interfaces, with built-in chat UI, streaming, and observability.

## What problem it solves

Building a polished chat interface for an LLM application from scratch requires significant frontend work; Chainlit provides a production-ready UI with a simple Python decorator API so you can focus on the model logic.
