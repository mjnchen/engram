# Gemma (Google)

## What it is

Lightweight open-weight model family from Google DeepMind built on the same research behind Gemini, released in 2B and 7B parameter sizes.

## What problem it solves

Researchers and developers need capable open models that run on consumer hardware; Gemma delivers Gemini-class techniques in a small, permissively licensed package.
