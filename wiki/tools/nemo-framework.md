# NeMo Framework

## What it is

NeMo Framework is an end-to-end framework by NVIDIA for building, training, and deploying generative AI models, covering data curation through fine-tuning and alignment.

## What problem it solves

Training production-grade LLMs requires tightly integrated tools for data pipelines, distributed training, and deployment; NeMo provides these as a cohesive stack optimized for NVIDIA hardware.
