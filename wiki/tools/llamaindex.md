# LlamaIndex

## What it is

LlamaIndex is a data framework by LlamaIndex for connecting LLMs to external data sources, enabling retrieval-augmented generation and structured query applications.

## What problem it solves

LLMs lack access to private or up-to-date data; LlamaIndex handles ingestion, indexing, and retrieval so you can query documents, databases, and APIs through an LLM interface.
