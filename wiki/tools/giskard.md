# Giskard

## What it is

Testing and evaluation library from Giskard AI for detecting bias, hallucinations, and security vulnerabilities in LLM applications.

## What problem it solves

LLM apps can fail in systematic ways (prompt injection, hallucination, bias) that unit tests miss; Giskard automates adversarial scanning to surface these issues before production.
