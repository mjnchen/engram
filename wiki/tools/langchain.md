# LangChain

## What it is

LangChain is an open-source framework by LangChain AI for composing LLM calls, tools, memory, and external data into chains and agents.

## What problem it solves

Building multi-step LLM applications requires wiring together many components (models, retrievers, tools, memory); LangChain provides standardized abstractions and pre-built integrations to do this quickly.
