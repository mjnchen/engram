# Langroid

## What it is

Langroid is a multi-agent LLM programming framework by Langroid with a message-passing architecture inspired by the Actor model, where agents communicate by sending and responding to messages.

## What problem it solves

Coordinating multiple specialized LLM agents requires a principled concurrency and communication model; Langroid's actor-style architecture makes agent interactions explicit and composable without complex orchestration code.
