# Haystack

## What it is

Haystack is an NLP framework by deepset for building production-grade RAG pipelines and question-answering systems with a composable, graph-based pipeline abstraction.

## What problem it solves

Building robust document retrieval and QA systems requires connecting document stores, retrievers, readers, and generators in configurable ways; Haystack provides tested components and a pipeline graph to wire them together.
