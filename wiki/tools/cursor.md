# Cursor

## What it is

AI-native code editor from Anysphere built on VS Code with deep LLM integration for inline code generation, multi-file editing, and natural language codebase Q&A.

## What problem it solves

Traditional editors treat AI assistance as a plugin afterthought; Cursor co-designs the editing experience around LLM capabilities, enabling agent-level code changes across an entire repository.
