# Dify

## What it is

Dify is an open-source LLM application development platform by LangGenius with a visual workflow builder, built-in RAG pipeline, and multi-model management UI.

## What problem it solves

Building LLM apps with RAG and tool use typically requires custom code for each integration; Dify provides a no-code/low-code interface that lets teams prototype and deploy AI workflows without deep ML engineering.
