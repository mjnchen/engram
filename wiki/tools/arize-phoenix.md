# Arize Phoenix

## What it is

Open-source ML observability tool from Arize AI for tracing, evaluating, and debugging LLM applications and RAG pipelines.

## What problem it solves

RAG and agent pipelines have many failure modes that are invisible without tracing; Phoenix surfaces retrieval quality, LLM outputs, and latency in a local or hosted UI.
