# Llama 3 (Meta)

## What it is

Meta's third-generation open-weight large language model series, ranging from 8B to 405B parameters, with instruction-tuned and base variants.

## What problem it solves

Provides researchers and developers access to frontier-quality models they can run locally, fine-tune, or deploy without API costs or data-sharing constraints.
