# OpenAI Evals

## What it is

Open-source framework by OpenAI for evaluating language models and LLM-powered products on built-in and custom task benchmarks.

## What problem it solves

Makes it easy to write, share, and run reproducible evals so teams can track model quality regressions and compare across model versions.
