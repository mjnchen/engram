# StarCoder2

## What it is

Open code generation model from the BigCode project trained on The Stack v2, available in 3B, 7B, and 15B sizes.

## What problem it solves

Provides a permissively licensed, state-of-the-art code LLM that developers can self-host, fine-tune, and deploy without usage restrictions common in proprietary code models.
