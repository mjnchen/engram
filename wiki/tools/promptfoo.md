# promptfoo

## What it is

CLI and library for systematically evaluating, testing, and comparing prompts and LLM models.

## What problem it solves

Ad-hoc prompt testing misses regressions; promptfoo provides a structured test harness with assertions, red-teaming, and side-by-side model comparisons.
