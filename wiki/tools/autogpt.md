# AutoGPT

## What it is

AutoGPT is an early autonomous GPT-4 agent by Significant Gravitas that generates and executes its own task list, using tools like web search and file I/O to pursue long-horizon goals without step-by-step human guidance.

## What problem it solves

LLMs are reactive; AutoGPT demonstrated that an LLM could drive itself through multi-step tasks by looping: generate next action, execute, observe result, repeat.
