# Open LLM Leaderboard

## What it is

Automated leaderboard by Hugging Face that tracks and ranks open-source LLM releases across standardized benchmarks.

## What problem it solves

Keeps the open-source LLM ecosystem honest by providing a consistent, publicly visible comparison point updated as new models are released.

## Notes

- Uses lm-evaluation-harness as its backend
- Benchmarks include ARC, HellaSwag, MMLU, TruthfulQA, Winogrande, GSM8K
- Free to submit any model hosted on Hugging Face Hub
