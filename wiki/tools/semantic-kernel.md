# Semantic Kernel

## What it is

Semantic Kernel is an SDK by Microsoft for integrating LLMs into enterprise applications by exposing prompts as callable functions with plugin and memory abstractions.

## What problem it solves

Connecting LLMs to existing business logic and data sources in a maintainable way requires structuring prompts, tool calls, and memory as composable components; Semantic Kernel provides this architecture in C#, Python, and Java.
